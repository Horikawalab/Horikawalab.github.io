## Policy / 運用方針

### English
- This website is an official academic document of the laboratory.
- All contents should be factual, accurate, and kept up to date.
- Publications and research results must be approved by the PI before being added.
- Personal opinions or unpublished results should not be included.

### 日本語
- 本ウェブサイトは研究室の**公式学術文書**として位置づけられます。
- 掲載内容は事実に基づき、常に最新の情報を保つことを原則とします。
- 論文・研究成果の追加は、必ずPI（研究室主宰者）の承認を得てから行ってください。
- 個人的意見や未公開研究成果の掲載は禁止します。
