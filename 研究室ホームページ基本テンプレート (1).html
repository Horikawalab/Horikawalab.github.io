README.md (Student Update Rules)

# Lab Website Maintenance Rules

## Basic Policy
- This website is an official academic document.
- Do NOT add personal opinions or unreviewed results.

## Update Responsibility
- News: assigned student (rotating, yearly)
- Publications: PI approval required
- Members: self-update allowed, factual only

## Commit Rules
- One change = one commit
- Commit message in English
  e.g. "Add 2025 conference paper"

## Prohibited
- Private email addresses
- Under-review papers without label
- Experimental data not yet published
